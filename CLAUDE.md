# Instruções do projeto — abaetefest-discounts

## Repositório padrão

Este repositório (`abaetefest-discounts`) é o destino padrão para qualquer alteração em `products.json` (adicionar, editar ou remover produtos da Amazon para o Abaetéfest).

Não usar a pasta "Captura Amazon para Abaetéfest" para essas atualizações, a menos que o usuário peça explicitamente.

## Referência de schema

Ver `SCHEMA.md` para a estrutura de `products.json` (campos obrigatórios e categorias válidas).
